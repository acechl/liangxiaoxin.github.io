记录遇到的兼容性问题

获取下一个兄弟节点 (nextElementSibling 和 nextSibling) 的兼容

/**
 * 封装了一个兼容各个浏览器的获得下一个兄弟姊妹标签节点的函数
 * @param element
 * @returns {element}
 */
function  getNextElement(element){
    // 进行能力检测  就是看当前的浏览器是否支持这个对象的属性或是方法
    if(element.nextElementSibling){ // 高版本的浏览器支持的
        return  element.nextElementSibling;
    }else {
        // 到了else里面就是IE8及以下
        var ele = element.nextSibling;//先拿到当前对象的下一个节点
        while(ele&&ele.nodeType!=1){
            ele = ele.nextSibling;//继续往后找
        }
        return ele;//返回到函数的外面
    }
}

同理，获取上一个兄弟节点 (previousElementSibling 和 previousSibling)的兼容也可。

addEventListener 和 attachEvent 的兼容

使用方式：

DOM对象.addEventListener(事件名称，处理函数，函数执行时机);

好处：可以重复事件的处理程序
