记录Javascript有趣之处

在Javascript世界中，万物皆对象。

学习之余，过于沉闷，在此记录一些Js好玩有趣的用法，有助于锻炼自我逻辑。

数字交换不声明中间变量的方法

两个数字之间做交换，我们一般的做法是先声明一个中间变量z，来看看下面这种奇葩的做法：

var a=1,b=2;a=[b,b=a][0];
console.log('a:'+a+',b:'+b);
//输出a:2,b:1

是不是很过瘾？有种豁然开朗的感觉。

js整数的操作

使用 |0 和 ~~可以将浮点转成整型且效率方面要比同类的parseInt,Math.round 要快,在处理像素及动画位移等效果的时候会很有用。 

var foo = (12.4 / 4.13) | 0;//结果为3
var bar = ~~(12.4 / 4.13);//结果为3

还有一个小技巧，就是 !! 2个叹号，快速转化为布尔值。

var eee="eee";
alert(!!eee)
